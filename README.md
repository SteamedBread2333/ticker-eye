# Ticker Eye

Chrome 扩展：页面内可拖拽浮窗，显示自选股票行情（价格、涨跌、委比、量比等），并展示 **资金流向**（特大 / 大 / 中 / 小单，数据来自东方财富）。

## 功能概览

- 行情约每 3 秒刷新；资金数据有缓存，不会高频请求
- 添加 / 删除自选、拖拽浮窗、调整顺序、点击复制代码；**Ask AI** 可复制当前行信息
- 支持 A 股、港股、美股（代码格式见下）

## 安装

1. 打开 `chrome://extensions/`，开启「开发者模式」
2. 「加载已解压的扩展程序」→ 选择本仓库目录（**无需**先执行 `npm install`，日常用扩展不依赖 Node）

若需重新生成图标，可执行 `npm ci` 后运行 `npm run generate-icons`。

## 使用

1. 点击扩展图标，在弹窗里添加代码（如 `562500`、`600000.SH`、`0700.HK`、`AAPL`）
2. 打开「在页面显示行情」，刷新网页即可看到浮窗

## 代码格式提示

- A 股：`600000.SH`、`000001.SZ`；6 位数字默认按 A 股处理  
- 港股：`0700.HK`；4–5 位数字默认港股  
- 美股：`AAPL` 等

## 数据说明

- 行情：腾讯 / 新浪等接口  
- 资金流向：东方财富（扩展后台请求；调试时在扩展 **Service Worker** 里看网络，页面 Network 里通常看不到）
