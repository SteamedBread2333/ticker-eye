# Ticker Eye - 股票行情助手

一个实时股票行情Chrome扩展，可拖拽浮窗显示，每3秒自动刷新。

## 功能特性

- ✅ 实时股票行情显示（每3秒刷新）
- ✅ 支持添加/删除股票代码
- ✅ 可拖拽浮窗，不影响页面操作
- ✅ 支持上下箭头调整排序
- ✅ 点击股票名称复制代码
- ✅ 使用腾讯API和新浪API获取数据（支持A股和港股）

## 安装方法

1. 打开Chrome浏览器，进入扩展管理页面：`chrome://extensions/`
2. 开启"开发者模式"（右上角开关）
3. 点击"加载已解压的扩展程序"
4. 选择本项目文件夹

## 使用方法

1. 点击浏览器工具栏中的扩展图标
2. 在弹窗中输入股票代码（如：`513120`, `600000.SH`, `000001.SZ`, `0700.HK`）
3. 点击"添加"按钮
4. 确保"在页面显示行情"开关已开启
5. 刷新当前页面，即可看到可拖拽的股票行情浮窗

### 浮窗操作

- **拖拽移动**：拖动标题栏可以移动浮窗位置
- **调整排序**：点击上下箭头按钮调整股票顺序
- **复制代码**：点击股票名称/代码即可复制到剪贴板
- **删除股票**：在popup中删除

## 支持的股票代码格式

- **A股**：`600000.SH`（上海）、`000001.SZ`（深圳）
- **港股**：`0700.HK`
- **美股**：`AAPL`、`TSLA`（仅支持新浪API）
- **简化输入**：6位数字（如`513120`）默认为A股，4-5位数字默认为港股

## 技术说明

- 使用腾讯API（qt.gtimg.cn）和新浪API（hq.sinajs.cn）获取股票数据
- 优先使用腾讯API，失败时自动切换到新浪API
- 支持A股、港股、美股（腾讯和新浪API都支持）
- 数据每3秒自动刷新一次
- 浮窗位置自动保存
